## Featured projects

### [Us.Vet.Imagens.Anonimizador](https://github.com/yasmine-vet-ads/Us.Vet.Imagens.Anonimizador)  
**Veterinary Ultrasound Image Anonymizer**

MVP open source for anonymizing veterinary ultrasound images using **Python, Streamlit and OpenCV**.

Created from a real need in veterinary diagnostic imaging: removing sensitive information from ultrasound images before using them in studies, presentations, teaching materials and technical documentation.

**Stack/topics:** Python · Streamlit · OpenCV · Image Processing · Data Privacy · Veterinary Ultrasound · VetTech

---

### [Ia-Vet-Doc](https://github.com/yasmine-vet-ads/Ia-Vet-Doc)  
**Veterinary Document Assistant**

Experimental project for organizing, searching and structuring veterinary documents such as reports, clinical histories and technical references using **Python, applied AI concepts, RAG and document processing**.

The project explores how technology can support clinical and technical information management without replacing professional review.

**Stack/topics:** Python · Applied AI · RAG · LLMs · Document Processing · Clinical Data · VetTech

---

### [BeeSpace](https://github.com/yasmine-vet-ads/BeeSpace)  
**Smart Beekeeping, Biodiversity and AgTech**

Project focused on biodiversity and environmental monitoring using bees as natural biosensors, connected to **IoT, satellite data, environmental indicators, computer vision and machine learning**.

**Stack/topics:** IoT · AgTech · Biodiversity · Satellite Data · Computer Vision · Machine Learning · ESG
